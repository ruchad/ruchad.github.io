Expiry Data Keeper

This app helps users make a list of items with expiry date.

Features:
* Expiry list conveniently displays the number of days left/past-due for each item.
* Assign categories to tasks.
* Items can be removed from the list by long clicking.
* The details page for each item, gives additional information about the item.
* Option to add reminder for tasks.
* Ability to sort tasks by categories or number of days left.
